# 🌐 Portfolio • Kuldeep Singh Rawat

A modern, responsive, and accessible **personal portfolio website** built with **HTML, CSS, and vanilla JavaScript**.  
It highlights projects, skills, education, and experience with smooth interactions and a clean UI.

---

## 🌟 Features
- **Responsive navigation** with scroll-based active link highlighting  
- **Mobile-friendly menu** with accessible toggle button  
- **Smooth scrolling** and scroll-aware header  
- **Hero section** with intro, CTA buttons, and profile photo  
- **About section** describing background & interests  
- **Skills grid** showcasing key technologies  
- **Projects showcase** with cards and links to details  
- **Education & Experience** in a timeline layout  
- **Contact form** with accessible inputs and labels  
- **Sticky footer** with dynamic copyright year  
- **Accessible design** (ARIA labels, focus states, reduced motion support)  
- **Theming with CSS variables** for easy customization  

---

## 🛠️ Technologies Used
- **HTML5** → Semantic structure, accessibility roles & labels  
- **CSS3** → Custom properties, Grid & Flexbox layouts, gradients, responsive design  
- **JavaScript (Vanilla)** → Navigation toggle, header scroll effects, IntersectionObserver for active links, keyboard accessibility
